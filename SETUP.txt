PROMPT — Starship
================================================================================

Config: ~/.config/starship.toml
Theme: orange/space, two-line layout

Line 1: ╭ ~/path  branch ±modified  took Xs  HH:MM
Line 2: ╰❯

Modules enabled:
  - directory       (bold yellow, truncates to 4 levels)
  - git_branch      (purple, shows branch name)
  - git_status      (blue, shows ⇡⇣ ahead/behind, ~modified, +staged, etc.)
  - python          (shows version + virtualenv when active)
  - golang          (shows version when in Go project)
  - nodejs          (shows version when in Node project)
  - rust            (shows version when in Rust project)
  - cmd_duration    (shows "took Xs" for commands over 2s)
  - jobs            (shows ✦ N when background jobs exist)
  - time            (dimmed purple, HH:MM)
  - character       (disabled — arrow is hardcoded in format)

================================================================================
MULTIPLEXER — tmux
================================================================================

Config: ~/.tmux.conf
Prefix: Ctrl+a (rarely needed — most things are no-prefix)

No-prefix keybinds:
  Ctrl+\        split right (new pane to the right)
  Ctrl+g        split down  (new pane below)
  Ctrl+t        new window
  Ctrl+n        next window
  Ctrl+p        previous window
  Ctrl+x        close pane (asks confirmation)
  Ctrl+h/j/k/l  move between panes (vim-style)
  Ctrl+y        open Codex AI in a split pane
  Ctrl+o        open Claude Code in a split pane
  Alt+h/j/k/l   (legacy — same as Ctrl, kept as fallback)

With prefix (Ctrl+a then):
  H/J/K/L       resize pane (repeatable)
  S             choose/switch session
  X             kill window
  r             reload tmux config
  |             split right
  -             split down

Visual:
  Status bar:       bottom, transparent background, orange text
  Active pane:      orange border + "● folder" label at top
  Inactive pane:    amber border + "○ folder" label
  Window names:     auto-renamed to current folder

fastfetch behavior:
  - Runs automatically when a NEW window opens (not on splits)
  - Exits cleanly if terminal is resized or split while running
  - Manual: type `fastfetch` anytime


================================================================================
FASTFETCH — Custom Python Curses Animation
================================================================================

File: ~/.config/fish/launch.py
Also at: ~/.config/fish/spaceship.py (standalone full-screen spaceship)

What it shows:
  - macOS Apple logo (ASCII, diagonal orange→grey→turquoise→purple gradient)
  - Spaceship orbiting the logo (toggles in front / behind each half-orbit)
  - Starfield across the full terminal (120 stars, gold/blue/purple, twinkling)
  - System info panel: OS, Kernel, Uptime, Shell, Terminal, WM, CPU, GPU,
    Memory, Disk, Display, Battery, Local IP, Packages
  - Progress bar + "any key to skip" (timed mode) or "any key to exit" (manual)

Memory calculation: active + wired + compressed pages (matches Activity Monitor)

Flags:
  --no-timeout    stay open until keypress (used when calling `fastfetch` manually)
  (no flag)       auto-exits after 8 seconds (used on new window open)


================================================================================
WINDOW MANAGER — AeroSpace
================================================================================

Config: ~/.config/aerospace/aerospace.toml
Style: i3-like tiling, 8px gaps

All shortcuts use Caps Lock as the trigger key.
Caps Lock is remapped via Karabiner to Ctrl+Option+Cmd when held.
Caps Lock tapped alone = Escape.

Keybinds (physical: Caps + key):
  Caps+h/j/k/l          focus window left/down/up/right
  Caps+Shift+h/j/k/l    move window left/down/up/right
  Caps+1..5             switch to workspace 1–5
  Caps+Shift+1..5       move focused window to workspace 1–5
  Caps+Enter            open iTerm
  Caps+e                open VS Code
  Caps+b                open Firefox
  Caps+/                rotate tile direction (horizontal ↔ vertical)
  Caps+Space            toggle float / tile
  Caps+f                fullscreen
  Caps+q                close focused window
  Caps+Tab              jump to last workspace
  Caps+=                resize +50
  Caps+-                resize -50
  Caps+r                reload AeroSpace config
  Caps (tap alone)       Escape

Workspaces: 1, 2, 3, 4, 5 (persistent)
Gaps: 8px inner, 8px outer

================================================================================
PYTHON — pyenv
================================================================================

pyenv manages Python versions independently of the system Python.
Config in ~/.config/fish/config.fish (pyenv init'd on shell start).

Useful commands:
  pyenv install 3.x.x     install a version
  pyenv global 3.x.x      set global default
  pyenv local 3.x.x       set version for current directory (.python-version)
  pyenv versions           list installed versions
  python3 -m venv .venv    create a virtual environment
  source .venv/bin/activate.fish   activate venv in fish
