# Neovim Workflow

Leader is `Space`.

## Navigation

- `-` or `Space e`: edit files with Oil
- `Space ff`: find files
- `Space fg`: grep text
- `Space fr`: recent files
- `Ctrl-h/j/k/l`: move between splits

## Git

- `Space gd`: Diffview working tree
- `Space gD`: Diffview staged changes
- `Space gh`: current file history
- `Space gH`: repository history
- `Space gq`: close Diffview
- `Space gl`: open Lazygit in a tab
- `]h` / `[h`: next/previous hunk
- `Space gp`: preview hunk
- `Space gs`: stage hunk
- `Space gr`: reset hunk

## LSP and formatting

- `gd`: definition
- `gr`: references
- `K`: hover
- `Space rn`: rename
- `Space ca`: code action
- `Space ld`: line diagnostics
- `Space lf`: format

## Copilot

- Run `:Copilot auth` once.
- `Ctrl-f`: accept suggestion
- `Ctrl-b` / `Ctrl-v`: next/previous suggestion
- `Ctrl-e`: dismiss suggestion
